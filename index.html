Feather Finder

A user-friendly environmental web app designed to help you discover and learn about the birds around you. If you want to know what birds live around you, Feather Finder has you covered. With a simple search, you can look up birds by location. This app uses the Ebird API, presenting the user with detailed information about each bird. Hackathon Project

Woody Click

An interactive web application inspired by the game Cookie Clicker. In Woody Click, your task is simple yet engaging - click on a virtual tree to accumulate points. The ultimate goal? To amass the highest score! To embark on this tree-clicking adventure, all you need is a Gmail account. Woody Click utilizes the convenient Google Firebase Authentication for seamless account creation. As you progress through the game, your point-earning potential will plateau. By making donations to the Team Trees initiative through the website, you can break free from the plateau. The Python web scraping program checks for matching donation and account names. When a match is detected, it unlocks perks such as increased trees per click and more trees per second. The bonuses give the user instant gratification and entice the user to donate more. Check out the GitHub repository here.

Technologies used include:

  • HTML
  • CSS
  • JavaScript
  • React
  • Python
  • Google Firebase

IOS Project

Babel Beauty

A handy iOS app that's like a beauty blog for cosmetics enthusiasts. Here's how it works: you can write articles about makeup, give ratings to different products, and share your thoughts with the community. Once you have an account, others can read your reviews and insights. This app is perfect for young adults and anyone keen on makeup, helping them explore and experiment with new cosmetics to level up their makeup game. BabelBeauty taps into the Makeup API, which displays a variety of makeup products complete with brand details and product types to the user. All the user accounts and blog posts are safely stored in the Google Firebase database, ensuring a smooth and secure experience. Check out the GitHub repository here and the YouTube demo here.

Technologies used include:

  • Swift
  • Google Firebase
